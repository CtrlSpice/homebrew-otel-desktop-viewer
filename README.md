# homebrew-otel-desktop-viewer

Homebrew tap for [otel-desktop-viewer](https://github.com/CtrlSpice/otel-desktop-viewer).

```bash
brew tap ctrlspice/otel-desktop-viewer
brew install --cask otel-desktop-viewer
```

The cask is generated by GoReleaser on release. Edit `Casks/otel-desktop-viewer.rb` manually only if you know what you're doing.
